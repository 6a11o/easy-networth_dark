# Easy Networth

A modern, interactive financial tracking application that helps users monitor and visualize their net worth over time. Easy Networth provides a comprehensive view of your financial health by tracking assets, liabilities, and net worth changes.

## Features

- **Dynamic Net Worth Tracking**: Monitor your overall financial health with real-time calculations of assets and liabilities.
- **Interactive Data Visualization**: View your net worth trends with responsive, color-coded charts that highlight positive and negative performance.
- **Time Period Selection**: Analyze your financial progress across different time periods (1M, 3M, 6M, 1Y, or All Time).
- **Account Management**: Add, edit, and categorize your financial accounts for a complete picture of your finances.
- **Currency Support**: Track your finances in multiple currencies.
- **Responsive Design**: Enjoy a seamless experience across desktop and mobile devices.

## Technology Stack

This project is built with:

- **React**: Front-end UI library
- **TypeScript**: Type-safe JavaScript
- **Vite**: Fast, modern build tool
- **shadcn/ui**: High-quality UI components
- **Tailwind CSS**: Utility-first CSS framework
- **Recharts**: Responsive charting library
- **React Router**: Navigation and routing
- **React Context API**: State management

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or bun

### Installation

```sh
# Clone the repository
git clone <repository-url>

# Navigate to the project directory
cd easy-networth_dark

# Install dependencies
npm install
# or
bun install

# Start the development server
npm run dev
# or
bun dev
```

The application will be available at http://localhost:5173 (or the port specified in your terminal).
